Skynet

Skynet is a sample Ethereum blockchain dApp where you can input a command which is simply saved to the blockchain and can be viewed later. The frontend is made with React JS and the contract is hosted on Ethereum Rinkeby testnet.

METAMASK LOGIN

You can use metamask to connect your wallet. Make sure to select a Rinkeby Ethereum testnet account.

VIEW CONTRACT STORAGE

After connecting your wallet, you can view all the previous commands saved in the contract.

CONTRACT INTERACTION

You can input a command and execute it which simply interacts with the contract to save it in storage.

VIEW CONTRACT STORAGE

The command you have entered is saved in the blockchain and can now be viewed.
