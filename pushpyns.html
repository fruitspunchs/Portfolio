pushpyns

Pushpyns is an Android app where you can save locations as Pyns. Business users can attach work orders to Pyns and get customer signatures to complete work orders. Please note the service has ended so you wont get past the login screen.

LOGIN SCREEN

Submit user credentials using AsyncTask and get a session ID.

MAIN SCREEN

Search locations with Google Maps to use as Pyns.

ADD PYN SCREEN

Save Pyn to server. Fields are filled with current search location. You can also add categories and images.

NAVIGATION DRAWER

You can choose to view Pyns, Work Orders, Terms & Conditions, or Log Out.

VIEW PYNS SCREEN

Uses SQLite with ContentProviders, CursorLoaders, and RecyclerViews to show a list of Pyns. You can sort, search, and filter by category.

VIEW PYN DETAILS

Here you can edit or delete Pyns. You can also get directions via Google Maps and share the Pyn via messaging or e-mail.
