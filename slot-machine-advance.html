Slot Machine Advance

Slot Machine Advance is an Android Slot Machine game written in Java with Eclipse. It used Bugsense for crash reporting, Flurry for analytics, and AdMob for monetization. The game uses open-source art edited with GIMP and open-source music edited with Audacity.

MAIN MENU SCREEN

Here you select game modes and themes. The game hasn't been updated in a while so leaderboards might not work.

FIVE REEL SLOT MACHINE

You can play a five-reel slot machine and play up to 25 lines. Online Jackpot is a PHP script that returns a JSON with a 1,000,000 value, but I used a free server and now its offline. It's the most popular game mode.

PORTRAIT MODE

You can rotate the screen to change between portrait and landscape mode anytime.

THREE REEL SLOT MACHINE

You can also play a three-reel slot machine and play up to 5 lines. It's the least popular game mode.

POKER MACHINE

Finally, theres a poker machine. It's the second most popular game mode.
